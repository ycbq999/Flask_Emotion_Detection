Facial Expression Application

Emotion Expression Classification: This App is driven by data and machine learning model. The Model has pretrained using kaggle dataset and uploaded to huggingface model hub. As soon as user upload the image, the app will connect to the hugging face api to connect to the model and show the probability of the prediction. The end prediction is the expression with the highest probability.
